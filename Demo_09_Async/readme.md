# Demo Async
Sénario -> Préparation d'un repas

## Méthodes
 - Café
    - Démarrer la machine (33s)
    - Service le café (12s)
 - Salade
    - Couper les 4 légumes (10s + 5s par légumes)
    - Préparer la salade (10s)
  	- Ajouter la vinaigrette (5s)
 - Repas
    - Chauffer la poele (21s)
    - Faire la cuire le repas (21s)
 - Service l'apéro (4s)
